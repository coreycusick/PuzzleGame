Purpose of using images

Adding images to a website is simple, easy, and makes the website look more attractive. Adding images allows the user to interact more and gather information quicker than reading bunch of text. If you have a step by step instruction, you could add images to help with understanding. However, there are some things to follow when adding images to a website.

  • Do not use high resolution photos on the website. If you use high res photos, the photos will slow down the site and take a long time to load

There are multiple different types of images that work well in specific instances. Sometimes you will use different file extensions for different purposes.

Types of Images

  • Vector Image - an image comprised of lines and curves that are based on mathmatical functions
  • Bitmap Image - an image that is comprised of pixels in which every pixel is marked with a different color. Due to the amount of pixels, bitmap images are considerably larger compared to vector images
    • Scalable Vector Graphics (SVG)
    An XML markup language that can be created using a basic text editor and knowledge of the SVG language
    Graphics Interchangeable format (GIF)
    This photo can be used to support transparent colors, can also be used for animated images. They have a limited color palette, they are typically not suitable for photos
    Joint Photographic Experts Group (JPEG or JPG)
    These types of photos can support up to 16M colors and is the most popular photo format. These can be compressed to save space compared to most bitmap images. JPGs do not support transparency or animations
    Portable Network Graphics (PNG)
    Created to replace GIFs, supports transparency and over 1M colors, this file can also be compressed, making it smaller than a GIF. They are usually used for logo designs because they support transparency

Images are measured in pixels. A pixel is the smallest unit of measurement on a computer and an image is made up of thousands to millions of pixels. Whenever we set a width or height, the pixel measurement will be used, unless otherwise stated.

Aligning images

You can align/position your images in a couple different ways. This allows the flow of the web page to be more cohesive for the user. Some pictures can align with a certain position on a site, while others may align itself with text. You can align your image by using the align attribute. Linking images

Linking images is very helpful when building a website. You can link a logo back to the homepage, link merchandise to a check out page, or just to make a photo gallery and link the photo to a larger image. Linking in general is extremely helpful, but linking images allows you to make the website a tad bit more interactive.
