HTML Hyperlinks

Absolute Links vs. Relative Links

Absolute Links are used when we link our web page to another website by using the URL provided to us. For Example https://www.w3schools.com

Relative links are used when we use direct paths or files on our computer and/or server to link web pages together. We do this to identify the correct file and location by mixing file names with forward slashes( /). Example: Index.html or webPortfolio/exercises/exerciseOne.html
