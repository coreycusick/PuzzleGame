All of the branches of Web Design

Web development can consist of multiple layers and the use of multiple languages, depending on how interactive you would like the webite to be. As a web developer, you are in charge of the:

  • Website design
  • Website Userability
  • Functionality of the Website
  • Deployment of the website
  • Copywriting
  • Search Engine Optimization
A web developer doesn't need to be able to do all of these things, but it does help to have knowledge of each area

Web Site Structure

When you create a website, you must have the correct skeleton for the webpage to run. This is also called a boiler plate. The boiler plate consists of the following tags to be recognized on the website on the computer.

  1. !DOCTYPE html
    • This indicates that the website is HTML5
  2. Head
    • This is where we put all of the scripting and the information that runs in the background.
    • This is where we place CSS and JavaScript external files
    • Meta data and Keywords also go here to help with Search Engine Optimization
  3. Title
    • This is where you place the title of your webpage/website
    • This information shows up in the browser tab
  4. Body
    • This is the information viewable on the website. Anything that you want your audience to see, you place here
    • This is where you place:
      • Text
      • Images
      • Videos
      • Tables
      • Forms
      • ETC.
